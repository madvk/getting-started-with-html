HTML

HTML stands for Hypertext Markup Language. HTML is made up of:
- Text: written content
- Markup or what the text looks like; ((markup is used to make it look different))
- May have references to other files, say images or videos
- May have links to other pages

Tags and Elements

HTML documents are made of HTML elements. When writing HTML, we tell browsers the type of each element by using HTML tags.
HTML tags are shown in angled brackets. Most consist of an opening tag (written as < tag >) and a closing tag (written as < tag />). Some do not have a closing tag and are called void tags.
Everything within a set of opening and closing tags is an element. This video explains more in detail, including the distinction between these and about the concept of an attribute as well.

Need to know this about Computers

Computers interpret instructions literally, and need to be told everything clearly and exactly. Programmers need to write exactly the way a computer understands, that is they need to write with correct syntax. A small mistake by a programmer can cause huge problems in a program.

Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them.
